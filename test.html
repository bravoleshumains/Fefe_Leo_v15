Cours Groupes-Anneaux-Corps

Chapitre 1: Groupes

Dans ce chapitre, nous introduisons les notions de fonctions holomorphes, de dérivées complexes et de formules de Cauchy.

Soit E un ensemble, on appelle une application * de (E,E)→E une loi de composition interne. L’appellation est limpide, c’est une loi qui permet de "mixer" des éléments d’un même ensemble entre eux.

Définition

On appelle (G,*) un groupe si G est un ensemble non vide et * est une L.C.I sur G vérifiant les trois propriétés suivantes:

  • ∀x, y, z ∈ G, x * (y * z) = (x * y) * z

  • ∃e ∈ G, ∀x ∈ G, x * e = e * x = x

  • ∀x ∈ G, ∃x − 1 ∈ G, x * x − 1 = x − 1 * x = e

Chapitre 2: Anneaux

Ce chapitre traite de l'intégration de fonctions complexes le long de chemins courbes, ainsi que du théorème de Cauchy pour les contours simples.

Définition

Soit A et B deux anneaux. Un morphisme f de A vers B est une application f : A → B telle que:

  • f(0) = 0 et f(1) = 1

  • ∀a, \Tilde{a} ∈ A, f(a + \Tilde{a}) = f(a) + f(\Tilde{a})

  • ∀a ∈ A, f( − a) = f(a)

  • ∀a, \Tilde{a} ∈ A, f(a*\Tilde{a}) = f(a)*f(\Tilde{a})

Chapitre 3: Corps Finis

Nous explorons les séries de Laurent et leur convergence, ainsi que les résidus et leurs applications.

Définition

Un corps (K, +, *) est un anneau dont tout élément non nul est inversible.

Exemple: ℚ, ℝ, ℂ

Définition

Soit (K, +, *) un corps. Une partie (P ⊂ K) est un sous-corps si c’est un sous-anneau tel que ∀x ∈ P, x − 1 ∈ P. On dit aussi que K est une extension de P.

Soit A un anneau. Soit n un entier naturel. On peut bien sûr le penser comme 1 + 1 + ⋯ + 1 (n fois). Mais alors il prend un sens dans A. De plus, si n est négatif, n =  − ( − n). On obtient ainsi un morphisme d’anneaux :

ι : ℤ → A

Autrement dit, ι(1) = 1, ι(2) = 1 + 1, ι(3) = 1 + 1 + 1 etc. Et ι( − 1) =  − ι(1), ι( − 2) =  − ι(2), ι( − 3) =  − ι(3) etc. Le noyau de ι est un idéal de A. Il s’écrit donc (n) pour un entier naturel n. L’entier n est appelé la caractéristique de A. On la note car(A).
